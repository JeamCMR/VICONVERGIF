# VICONVERGIF

El codigo es de uso libre.

Para la ejecucion se debe tener instalado Python.
Se debe instalar las librerias del archivo requirements.txt  con el comando pip install -r requirements.txt

Instruccion de uso.

1) Abrimos la consola de preferencia
2) Ejecutado la siguiente instruccion python convergif.py nombre_video.mp4 --salida nombre_gif.gif --fps "Cantidad de fps por defecto 10" --ancho "Tamaño del año del gif por defecto 400px"
Ejemplo :
python convergif.py prueba.mp4 --salida prueba2.gif --fps 10 --ancho 640
